Detailed Description

This page gets expanded when a item of playersList in WerewolfMainPage is pressed. Every field is initally filled with the player's current details. The player is modified when the "Modify Player" gets pressed.

Property Documentation

player : var

The player who's currently selected, null if no player is selected.


Method Documentation

close()

Collapses the ModifyPlayerPage and resets it, sets expanded to false.


openWithPlayer(int playerId)

Expands the ModifyPlayerPage and loads the player's details, sets expanded to true.


prepare()

Loads users data into the input fields.
