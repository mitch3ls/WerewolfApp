Detailed Description

This page gets expanded when the FloatingActionButton in WerewolfMainPage is pushed. Every field is initally empty until the user fills them. A new player is created when the "Create" button gets pressed.

Method Documentation

close()

Collapses and resets the AddPlayerPage, sets expanded to false.


open()

Expands the AddPlayerPage, sets expanded to true.
