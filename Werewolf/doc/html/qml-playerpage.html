Detailed Description

The PlayerPage contains ever element you see when the AddPlayerPage or the ModifyPlayerPage is expanded. It also processes the input and directly calls submit with the player object (without the playerId).

Property Documentation

expanded : bool

Describes whether the PlayerPage is expanded or not.


name : alias

The text in the name AppTextField.


notes : alias

The text in the notes AppTextEdit.


role : alias

The role in the roleChooser.


roleText : alias

The text in the role button. ("Choose role" if not specified)


title : string

The text in the submit button. ("Create" or "Modify Player")


Signal Documentation

submit(var player)

Submits created player object to derived component.

When the user presses the submit button ("Create" or "Modify Player") a player object is created and passed to the derived component's signalHandler.


Method Documentation

reset()

Resets the input fields and hides the roleChooser.

Restores default values in all text inputs and hides the errorField and the roleChooser. Calls the roleChooser's reset method.
